# FableTask
# App de Lista de Tareas Diarias 📝

Este proyecto es una aplicación iOS desarrollada en **Swift** con **Xcode**.  
La app permite a los usuarios organizar y gestionar sus tareas diarias de manera sencilla.

---

## 📱 Características (planeadas)
- Crear, editar y eliminar tareas.
- Marcar tareas como completadas.
- Interfaz simple y fácil de usar.

---

## 🚀 Tecnologías utilizadas
- Swift
- Xcode
- UIKit (posiblemente SwiftUI en el futuro)

---

## 📂 Estructura del proyecto
- `AppDelegate.swift` – configuración inicial de la app.
- `SceneDelegate.swift` – manejo de la ventana principal.
- `ViewController.swift` – controlador principal de la interfaz.
- `Main.storyboard` – diseño de la pantalla inicial.

---
